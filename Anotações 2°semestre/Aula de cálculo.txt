O gráfico cria oportunidade para vizualizar os dados de forma mais concisa independente da forma do gráfico. Todo gráfico precisa de um contexto por trás para ser descrtivo e entendido. Retroalimentação é usada a partir do tratamento de dados para criar gráficos que tem itens e dados em comuns.
