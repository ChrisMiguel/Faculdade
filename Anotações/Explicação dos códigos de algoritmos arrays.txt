O que é push () em JavaScript?
O método push() adiciona um ou mais 
elementos ao final de um array e retorna o novo comprimento desse array

var lista_mensagem = []
lista_mensagem.push



length (tamanho, em português) é uma propriedade dos arrays
 em JavaScript que retorna ou define o número de elementos em um determinado array.

O que é o indexOf?
indexOf() O método indexOf() retorna o 
primeiro índice em que o elemento pode ser encontrado no array, retorna -1 caso o mesmo não esteja presente.
